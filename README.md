# PyCalc: Dynamic Theme Calculator

PyCalc is a clean, functional arithmetic tool developed using Python's **Tkinter** library. Beyond standard calculation capabilities, this project focuses on **dynamic UI state management**, allowing users to switch seamlessly between Light and Dark visual themes.

---

## 🌓 Modern Theme Toggle
The standout feature of this application is the custom-built theme switcher. 
* **Light Mode:** High-visibility interface with soft blue accents.
* **Dark Mode:** Eye-strain-reducing palette with high-contrast amber highlights.

## 🛠️ Technical Features

* **Grid Geometry Management:** Precision placement of buttons using Tkinter's `.grid()` system for a responsive, desktop-native feel.
* **Functional Logic:** Handles basic operations (Addition, Subtraction, Multiplication, Division) with error handling for edge cases.
* **Global Styling System:** A centralized function that re-renders the background and foreground colors of all widgets instantly when the theme is toggled.
* **Custom Assets:** Integrated custom iconography for the theme toggle switch.
